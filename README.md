# Baby Tracker MVP

Минимальный веб-сервис для отслеживания:
- последнего кормления
- начала сна
- пробуждения
- истории событий за день

## Стек
- Next.js 15
- React 19
- Tailwind CSS
- Supabase
- Vercel

## Что уже есть
- Главный экран с 2 статус-карточками
- Кнопки `Начать кормление / Завершить кормление`
- Кнопки `Уложили спать / Проснулся`
- История событий за сегодня
- SQL-схема для Supabase

## Что стоит добавить следующим этапом
- Авторизацию по magic link
- Совместный доступ для мамы и папы
- Редактирование записей
- Выбор типа кормления и объёма
- Напоминания и push
- PWA и установка на домашний экран

## Локальный запуск

```bash
npm install
cp .env.example .env.local
# заполнить NEXT_PUBLIC_SUPABASE_URL и NEXT_PUBLIC_SUPABASE_ANON_KEY
npm run dev
```

## Настройка Supabase
1. Создайте проект в Supabase.
2. Откройте SQL Editor.
3. Выполните содержимое `supabase/schema.sql`.
4. В `children` должна появиться запись `Малыш`.
5. Скопируйте URL и anon key в `.env.local`.

## Деплой на Vercel
1. Залейте проект в GitHub.
2. Импортируйте репозиторий в Vercel.
3. Добавьте env-переменные из `.env.local`.
4. Нажмите Deploy.
